## Quick Start

Install Nix

```console
curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | sh -s -- install
```

Perform initial setup with:

```console
> direnv allow
> dev setup
```

Then explore other options with:

```console
> dev --help
```
